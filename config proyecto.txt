#Requisitos
Python 3.11 o superior
Mysql 8.0 o superior

configurar settings.py de 'proyecto' segun tu usuario, contra y la base de datos que uses, no creo que haga falta decir,
pero debes configurar la contraseña de root, y crear una base de datos, una vez creada debes configurar esto en settings.py
en mi caso lo configura asi:

# Database
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'tienda', #(nombre de base de datos)
        'USER': 'root',   #(nombre de usuario si deseas crear otro que tenga privilegios de admin, yo lo dejaria default)  
        'PASSWORD': 'Emilioejro2004', #(tu contraseña)
        'HOST': 'localhost', #(como es una base local usaremos localhost)
        'PORT': '3306', #(puerto de mysql)
    }
}
